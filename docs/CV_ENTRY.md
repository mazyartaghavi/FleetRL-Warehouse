# CV and Portfolio Entry

## Project title

FleetRL-Warehouse: Cooperative Multi-Robot Reinforcement Learning with ROS 2 and Gazebo

## One-line CV entry

Developed a reproducible cooperative warehouse-robotics platform combining custom multi-agent Q-learning/SARSA, PPO, A2C, DDPG, and SAC environments with ROS 2 fleet coordination, Gazebo simulation, lidar-based safety supervision, automated testing, and policy deployment.

## Three CV bullets

- Designed a multi-robot warehouse environment with shared rewards, decentralized observations, dynamic task allocation, pickup/delivery operations, collision handling, and reproducible evaluation metrics.
- Implemented independent Q-learning and SARSA from first principles and integrated Stable-Baselines3 PPO, A2C, DDPG, and SAC through discrete and continuous Gymnasium interfaces.
- Built a ROS 2 Jazzy/Gazebo Harmonic deployment stack with fleet-state communication, policy executors, differential-drive robots, lidar safety filtering, launch automation, unit tests, Docker assets, and GitHub Actions.
